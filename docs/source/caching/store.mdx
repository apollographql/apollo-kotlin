

## Interacting with the cache

To manipulate the cache directly, `ApolloStore` exposes `readOperation`/`readFragment` and `writeOperation`/`writeFragment` methods:

```kotlin
  // Reading data from the store
  val data = apolloClient.apolloStore.readOperation(BookWithAuthorName())

  // Create data to write
  val data = BookWithAuthorName.Data(
    id = "book1",
    title = "Les guerriers du silence",
    author = BookWithAuthorName.Author(
      id = "author1",
      name = "Pierre Bordage"
    )
  )
  // Write to the store. All watchers will be notified
  apolloClient.apolloStore.writeOperation(BookWithAuthorName(), data)
```
