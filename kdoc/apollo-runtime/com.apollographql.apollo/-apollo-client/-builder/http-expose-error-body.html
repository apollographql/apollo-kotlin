httpExposeErrorBody

fun httpExposeErrorBody(httpExposeErrorBody: Boolean?): ApolloClient.Builder

Configures whether to expose the error body in ApolloHttpException.

If you're setting this to true, you must read ApolloResponse.exception and close the body explicitly in case of an ApolloHttpException to avoid sockets and other resources leaking.

This is a convenience function that configures the underlying HttpNetworkTransport. See also networkTransport for more customization.
