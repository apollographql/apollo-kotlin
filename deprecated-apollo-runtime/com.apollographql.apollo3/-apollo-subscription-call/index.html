interface ApolloSubscriptionCall<D : Operation.Data> : Cancelable

ApolloSubscriptionCall is an abstraction for a request that has been prepared for subscription. ApolloSubscriptionCall`` cannot be executed twice, though it can be cancelled. Any updates pushed by server related to provided subscription will be notified via [Callback]

In order to execute the request again, call the ApolloSubscriptionCall.clone method which creates a new ApolloSubscriptionCall object.
