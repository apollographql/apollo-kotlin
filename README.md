  <h1 align="center">Apollo Kotlin</h1>
 
**The industry-leading GraphQL client for Kotlin**. Run on Android and all the Kotlin multiplatform targets. Apollo Kotlin delivers powerful caching, code generation, intuitive APIs, and comprehensive developer tools to accelerate your app development.
</header>


## ‚ùì Why Choose Apollo Kotlin?

 ‚úÖ 100% type safety, from your server to your app, using code generation.
 ‚úÖ Intuitive, powerful caching - intelligent in-memory or SQLite caching out of the box<br>
 ‚úÖ Excellent support for modern Android and Kotlin Multiplatform apps<br>
 ‚úÖ Always up-to-date - we prioritize support for the latest versions of GraphQL, Kotlin, Gradle, and more<br>
 ‚úÖ GraphOS ready - turnkey support for Persisted Queries, and `@defer`<br>
 ‚úÖ Production-tested - Powers countless apps worldwide that serve millions of end users<br>
