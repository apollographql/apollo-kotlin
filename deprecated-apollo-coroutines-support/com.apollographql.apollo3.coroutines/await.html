suspend fun <D : Operation.Data> ApolloCall<D>.await(): ApolloResponse<D>
Content copied to clipboard

Suspends the ApolloCall until it completes and returns the value on success or throws an exception on failure. The ApolloCall is cancelled when the coroutine running the operation is cancelled as well.

This is a convenience method that will only return the first value emitted. If more than one response is required, for an example to retrieve cached and network response, use toFlow instead.

the response on success.

the value type.
